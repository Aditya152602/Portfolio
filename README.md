## 🚀 Getting Started

### View Live

👉 [**aditya152602.github.io/Portfolio**](https://aditya152602.github.io/Portfolio)

### Run Locally

```bash
# 1. Clone the repository
git clone https://github.com/Aditya152602/Portfolio.git

# 2. Navigate into the folder
cd Portfolio

# 3. Open in browser — no build step needed!
open index.html

# OR serve locally with Python
python -m http.server 8000
# → Visit http://localhost:8000
```

> ✅ **Zero dependencies. Zero build tools. Pure HTML + CSS + JS.**
